# ES6 boilerplate webpack setup

## Setup

Goto folder

- use npm

```sh
npm install
```

- use yarn

```sh
yarn install
```

## Build

- Dev environment

```sh
npm run dev
```

- Production environment

```sh
npm run build
```
## Notes
- For dev environment, please use `http://localhost:8082/` url in the browser.
- Build files location `/public`
